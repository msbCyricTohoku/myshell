# myshell

A minimal C shell.

## Features
- * Infinite Pipelining
- * Interactive Autocomplete
- * Ghost Autosuggestions
- * Advanced Tokenizer
- * Logical Flow & Background Tasks
- * Intra-line Editing
- * Wildcard Globbing
- * Environment Variables
- * I/O Redirection

## Installation

Compile with standard C compiler (`gcc`) and `make`.

1. Clone the repository and compile:
   ```bash
   make
   ```

## Install it system-wide (requires root):

`sudo make install`

## Theming

To change the shell colors, open myshell.c and change the THEME macro at the top:

1: Ocean (Cyan / Blue)

2: Hacker (Matrix Green)

3: Sunset (Magenta / Yellow)

## Uninstallation

`sudo make uninstall`
